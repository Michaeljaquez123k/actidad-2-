Tema 1 - Texto

Propiedad Valor Inicial Descripción
color depends on the browser/user agent Define el color del texto. El valor puede ser un color en formato hexadecimal, RGB, HSL o con nombres de colores predefinidos.
font-family depends on the browser Establece la familia de fuentes para el texto. Puede ser una lista de fuentes separadas por comas, incluyendo fuentes específicas o genéricas.
font-size medium Define el tamaño de la fuente del texto. Puede expresarse en unidades absolutas (px, pt) o relativas (em, rem, %).
font-weight normal Establece el grosor del texto. Los valores comunes son normal, bold, o valores numéricos que van de 100 a 900.
font-style normal Determina el estilo de la fuente, como normal, italic, o oblique.
font-variant normal Controla la variación del texto, como small-caps para convertir el texto en mayúsculas pequeñas.
font depends on the browser Es un shorthand que permite definir todas las propiedades relacionadas con la fuente (font-style, font-variant, font-weight, font-size, line-height, y font-family) en una sola línea.
text-align start o left dependiendo del lenguaje Controla la alineación horizontal del texto dentro de su contenedor. Los valores incluyen left, right, center, justify, y más.
line-height normal Establece la altura de la línea del texto. Puede expresarse en números, porcentajes, o unidades de longitud.
text-decoration none Controla las decoraciones del texto, como subrayado (underline), tachado (line-through), o ninguno (none).
text-transform none Manipula la capitalización del texto. Valores incluyen uppercase (mayúsculas), lowercase (minúsculas), y capitalize (primera letra en mayúscula).
vertical-align baseline Controla la alineación vertical de un elemento en relación a su contenedor. Usado comúnmente para imágenes y texto.
text-indent 0 Define la sangría de la primera línea de texto. Puede expresarse en unidades de longitud o porcentajes.
letter-spacing normal Ajusta el espacio entre las letras. Puede ser un valor positivo (más espaciado) o negativo (menos espaciado).
word-spacing normal Controla el espacio entre palabras. Similar a letter-spacing, pero para palabras completas.
white-space normal Controla cómo se maneja el espacio en blanco dentro de un elemento. Valores comunes incluyen normal, nowrap, y pre.

Tema 2 - Enlaces

Característica Propiedades Descripción
Tamaño, Color y Decoración font-size, color, text-decoration
  • Tamaño: El tamaño de la fuente del enlace puede ser ajustado con font-size.
  • Color: El color del enlace se define con color, que puede variar dependiendo del estado del enlace (visitado, hover, etc.).
  • Decoración: La decoración, como el subrayado, se controla con text-decoration. Por defecto, los enlaces suelen estar subrayados (underline).
Decoración Personalizada text-decoration, :hover, :active, :visited Permite personalizar la apariencia de los enlaces en diferentes estados (hover, active, visited) usando pseudo-clases. La propiedad text-decoration permite agregar o quitar decoraciones como subrayado, líneas superiores o inferiores, entre otras.

Tema 3 - Imágenes

Característica Propiedades Descripción
Anchura y Altura width, height Define las dimensiones de la imagen. Se puede usar valores absolutos (px, cm) o relativos (%). Mantener la proporción de la imagen es crucial, por lo que generalmente se define solo uno de estos atributos o ambos con proporción.
Eliminar el Borde border, border-style Las imágenes dentro de enlaces pueden tener un borde por defecto en algunos navegadores. Se elimina configurando border o border-style a none.
